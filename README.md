# RelChat - AI-Powered Chat Application

RelChat is a full-stack chat application powered by a React frontend, FastAPI backend, and GROQ for LLM chat interactions. It uses Retrieval-Augmented Generation (RAG) for document uploads to enhance responses based on the content of uploaded files.
